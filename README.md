# dados-bgt

  Lanzador de dados, util para probar si estás jugando rol o necesitas dados virtuales para juegos de mesa.
  Hecho con bgt (Blastbay Game Toolkit) sin parte gráfica.

# teclas rápidas:

* espacio: para hacer una tirada.
* flechas izquierda y derecha: para modificar la cantidad de dados a lanzar.
* flechas arriba y abajo: para modificar las caras de los dados (pueden ser de 4, 6, 8, 10, 12, 20 y 100 caras).
* enter: para repetir última tirada.
* escape: para cerrar.
